# Supply-Chain-Analytics-Dashboard-Excel

## Project Overview 

This project presents an interactive Supply Chain Analytics Dashboard
built in Microsoft Excel using Power Query, Power Pivot, DAX, Pivot
Tables, and Pivot Charts. The goal is to transform raw transactional
data into actionable business insights through a star schema data model
and an interactive dashboard.

## Dataset

-   Dataset: DataCo Supply Chain Dataset
-   Approximate Size: 180,000+ records
-   Domain: Supply Chain & Logistics 

## Tools Used

-   Microsoft Excel
-   Power Query
-   Power Pivot
-   DAX
-   Pivot Tables
-   Pivot Charts

## Data Preparation

-   Cleaned and transformed raw data using Power Query.
-   Built dimension tables (Product, Customer, Date).
-   Created relationships using a star schema.
-   Loaded tables into the Excel Data Model.
-   Developed reusable DAX measures for KPIs.

## KPIs

-   Total Sales
-   Total Profit
-   Profit Margin %
-   Late Delivery %
-   Total Orders

## Dashboard Features

-   Executive KPI Cards
-   Monthly Sales Analysis
-   Product Analysis
-   Customer Segment Sales Analysis
-   Shipping Mode Analysis
-   Interactive Slicers and Filters
